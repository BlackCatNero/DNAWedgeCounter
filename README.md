# 듀엣 나이트 어비스 - 금쐐기 제작 현황 (WedgeCounter)

간단한 웹 앱으로 로컬에서 `index.html`을 열어 사용합니다.

기능 요약:
- 쐐기 추가: 이름과 하위(보라) 쐐기 필요량(5 또는 3)을 선택 후 추가
- 목록: 이름 / 보유 수 / 설계도(설계도 보유 수) / 하위(보라) 쐐기 보유 수 / 하위 필요 / 강화 가능 수치
- 보유 수, 설계도, 하위 수는 직접 입력 가능
- 각 항목에 `제작` 버튼이 있으며, 눌렀을 때 제작 로직이 실행됩니다.
	- 자원이 충분하면: 보유 수 +1, 설계도 -1, 하위 쐐기 -필요량.
	- 자원이 부족하면 경고창이 뜨며 다음 선택지를 제공합니다:
		1) `소모 없이 진행`: 보유 수만 증가, 자원 변화 없음.
		2) `강제 소모(음수 허용)`: 보유 수 증가, 자원을 강제로 소모하여 음수가 될 수 있습니다.
		3) `취소`: 아무 동작도 하지 않습니다.

사용법:
1. 이 폴더의 `index.html`을 더블클릭하거나 브라우저로 엽니다.
2. "쐐기 추가" 버튼으로 항목을 추가합니다.
3. 각 항목의 입력창에서 숫자를 수정하거나 + / - 버튼으로 조작합니다.

저장: 브라우저 `localStorage`에 저장됩니다.

필요한 개선 아이디어:
- 설계도/하위 쐐기를 제작 시 자동 계산 기능 추가
- CSV로 내보내기 / 가져오기
- 모바일 레이아웃 개선
